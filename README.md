# TicketBooking

# 🎟️ Online Ticket Booking Application

A full-stack web application built using **Spring Boot** that allows users to book tickets, store booking details, and generate travel information in PDF format.

---

## 🚀 Features

* 🧾 User ticket booking system
* 💾 Data stored in database
* 📄 Automatic PDF generation with booking details
* 🔍 Retrieve booking details using PNR number
* 🔗 REST API integration
* ⚙️ Backend developed using Spring Boot (MVC Architecture)

---

## 🛠️ Tech Stack

**Frontend:**

* HTML
* CSS
* JavaScript

**Backend:**

* Java
* Spring Boot

**Database:**

* MySQL

**Tools & Technologies:**

* REST APIs
* Docker (basic usage)

---

## 📌 How It Works

1. User enters booking details
2. Data is stored in the database
3. PDF is generated with user travel information
4. User can retrieve details using PNR number

---

## ▶️ How to Run the Project

1. Clone the repository

```bash
git clone https://github.com/Bharath508-CSE/TicketBooking.git
```

2. Open in your IDE (IntelliJ / Eclipse)

3. Configure MySQL database in `application.properties`

4. Run the Spring Boot application

5. Open browser and access the application
